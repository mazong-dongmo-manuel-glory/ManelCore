## Aperçu du projet

ManelCore utilise des technologies de pointe pour automatiser les processus d'affaires :
- **Backend** : FastAPI, LangGraph, LangChain.
- **Frontend** : Flutter (Desktop/Web).
- **Base de données** : Neo4j (Graphe) pour une gestion relationnelle complexe.
- **IA** : Intégration avec des LLM via OpenAI API (OpenRouter, LM Studio).
- **Veille Web** : API SEAO et crawler HTTP léger pour les pages publiques LinkedIn/Indeed.
